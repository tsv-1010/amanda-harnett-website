üÜò Common Issues & Solutions
Issue: Images not showing

Solution:

  • Check image path in src="" matches your folder structure
  • Make sure image file actually exists in that folder
  • Use forward slashes: assets/hero/image.jpg (not backslashes)
  • Refresh browser and clear cache (Ctrl+Shift+Delete)
Issue: Text looks broken or styling weird

Solution:

  • Make sure styles.css is in the same folder as index.html
  • Don't delete or rename CSS file
  • Refresh browser and clear cache
  • Check browser console for errors (F12)
Issue: Mobile menu not working

Solution:

  • Make sure script.js is in same folder as index.html
  • On mobile (width < 768px), hamburger should appear
  • Check F12 console for JavaScript errors
  • Try different browser to isolate issue
