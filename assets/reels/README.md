## Video Specifications

- **Aspect Ratio:** 9:16 (vertical/portrait)
- **Recommended Resolution:** 1080x1920 (or 720x1280 for smaller file size)
- **Format:** MP4 (H.264 codec)
- **Max File Size:** Keep under 10MB each for fast loading
- **Duration:** 15-60 seconds works best

## Tips for Optimization

1. Use HandBrake or similar to compress videos
2. Target bitrate: 2-4 Mbps for good quality/size balance
3. Poster images help with initial load appearance
4. Use first frame of video as poster for seamless experience
